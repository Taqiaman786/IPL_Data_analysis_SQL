# IPL_Data_analysis_SQL
🏏 Cricket Data Analysis (SQL Project)
📌 Overview

This project analyzes cricket match data using SQL to uncover key insights about team performance, match outcomes, and player achievements.

The dataset was sourced from Kaggle and processed using MySQL to demonstrate real-world data analysis skills.

📂 Dataset
Source: Kaggle
File: Cricket_data.csv
Contains match-level data including:
Teams (home & away)
Match results
Toss decisions
Scores (1st & 2nd innings)
Player of the Match
Venue details
🛠️ Tech Stack
SQL (MySQL)
Data Cleaning & Transformation
Aggregations & Joins
Analytical Queries
🧹 Data Preparation

Key preprocessing steps:

Converted date column into proper DATE format
Cleaned inconsistent values
Ensured data integrity for analysis
📊 Key Insights
🏆 Team Performance
Identified top-performing teams based on total wins
Compared home vs away match distribution
🎯 Toss Impact
Analyzed how often toss-winning teams win matches
Studied decision patterns (bat vs bowl)
🔥 Match Analysis
Found highest and lowest first-innings scores
Identified highest successful run chase
Compared chasing vs defending success rates
🌍 Venue Insights
Highlighted venues hosting the most matches
⭐ Player Performance
Identified players with most "Player of the Match" awards
Ranked top performers
